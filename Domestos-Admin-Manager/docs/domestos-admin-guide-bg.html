6. Как се определя класирането (за Domestos и оценените локации)

Не подреждаме само по звезди — една единствена 5★ оценка не е толкова надеждна, колкото 4,5★ от 20 души. Затова използваме претеглена оценка (същия метод като IMDb „Топ 250“): локациите с малко оценки се изтеглят към средното.

точки = ( v / (v + m) ) · R  +  ( m / (v + m) ) · C

R — среден рейтинг · v — брой оценки · m — праг на доверие (5) · C — средният рейтинг за всички локации.

Пример (при C = 4,0 и m = 5):   4,5★ от 20 оценки → 4,40 точки 🥇  |  5,0★ от 1 оценка → 4,17 точки. Така доказаната локация изпреварва тази с една случайна оценка.

  • Преди старта на кампанията (29.06) показателите „в периода“ и значките за участие са нулеви — това е нормално.
